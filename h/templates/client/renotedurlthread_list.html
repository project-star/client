How to get started

  1. To create highlights, annotations, video snips and audio clips install ReNoted browser extension from the Chrome Playstore

  2. Once you have created some annotations, login into the ReNoted Web application to view them.

  3. Each card represents one Web page. The card title can be clicked to navigate to the corresponding Web page. By default, only one annotation from the Web page is shown.

  4. Each card contains one or more annotations, shown by the badge next to the title. To view all the annotations on a page click the expand button . Similarly, collapse button can be used to collapse the card to default view.

  5. Annotations inside a card, can be Edited or Deleted

    Use Edit to add, delete or modify Tags and notes in Text annotations. In case of Media annotations, Edit allows you to also modify the Start and End times of the Snips and Clips.

  6. Use the powerful Search to find annotations with plain text queries or filter by "tag:" and "type:"
